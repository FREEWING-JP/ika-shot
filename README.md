ika-shot
=========
スプラトゥーンの勝敗を記録するWebアプリケーションです。

## 説明
[ika-camera](https://github.com/hogesuke/ika-camera)を使用してスプラトゥーンの結果画面をカメラで検出し、勝敗を自動記録します。
ika-shotは[ika-camera](https://github.com/hogesuke/ika-camera)で記録した内容を閲覧するためのWebアプリケーションです。
